Detailed Description

Bit manipulation macros.

Author:
Per Lundberg <per@chaosdev.org>

Henrik Hallin <hal@chaosdev.org>

Vladimir Sorokin

Version:
1.5
Date:
1998-2004
This file is endian-specific and needs to be rewritten for big-endian when we port to such a CPU. It provides very convenient functions for reading and writing bit values to a integers, as well as reading the upper/lower half of a 32-bit and 16-bit integer.
