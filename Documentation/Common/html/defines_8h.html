#define PACKED   __attribute__ ((packed))
 

Use this define to tell the compiler that a structure should be packed.

A packed structure uses no padding, which may otherwise be put in by the compiler to get a certain alignment on the data. For some structures that are locked to a specific format (for example a binary file format), such practices are directly harmful and you need to mark such structures as PACKED for the code to work properly..
