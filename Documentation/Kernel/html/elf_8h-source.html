00455 /*
00456 A This means the addend used to compute the value of the relocatable field.
00457 
00458 B This means the base address at which a shared object has been loaded into memory
00459 during execution. Generally, a shared object file is built with a 0 base virtual address,
00460 but the execution address will be different.
00461 
00462 G This means the offset into the global offset table at which the address of the
00463 relocation entry's symbol will reside during execution. See "Global Offset Table''
00464 below for more information.
00465 
00466 GOT This means the address of the global offset table. See "Global Offset Table'' below
00467 for more information.
00468 
00469 L This means the place (section offset or address) of the procedure linkage table entry
00470 for a symbol. A procedure linkage table entry redirects a function call to the proper
00471 destination. The link editor builds the initial procedure linkage table, and the
00472 dynamic linker modifies the entries during execution. See "Procedure Linkage
00473 Table'' below for more information.
00474 
00475 P This means the place (section offset or address) of the storage unit being relocated
00476 (computed using r_offset.
00477 
00478 S This means the value of the symbol whose index resides in the relocation entry.
00479 */
