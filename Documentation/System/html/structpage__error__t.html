Field Documentation

int page_error_t::p
 

bit(0) P

0 The fault was caused by a nonpresent page. 1 The fault was caused by a page-level protection violation.

int page_error_t::w_r
 

bit(1) W/R

0 The access causing the fault was a read. 1 The access causing the fault was a write.

int page_error_t::u_s
 

bit(2) U/S

0 The access causing the fault originated when the processor was executing in supervisor mode. 1 The access causing the fault originated when the processor was executing in user mode.

int page_error_t::rsvd
 

bit(3) RSVD

0 The fault was not caused by a reserved bit violation. 1 The page fault occured because a 1 was detected in one of the reserved bit positions of a page table entry or directory entry that was marked present.
