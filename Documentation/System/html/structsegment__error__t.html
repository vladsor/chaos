Detailed Description

Error code pushed onto the stack for segment exceptions.

Field Documentation

int segment_error_t::ext
 

EXT External event (bit 0).

When set, indicates that an event external to the program caused the exception, such as a hardware interrupt.

int segment_error_t::idt
 

IDT Descriptor location (bit 1).

When set, indicates that the index portion of the error code refers to a gate descriptor in the IDT; when clear, indicates that the index refers to a descriptor in the GDT or the current LDT.

int segment_error_t::ti
 

TI GDT/LDT (bit 2).

Only used when the IDT flag is clear. When set, the TI flag indicates that the index portion of the error code refers to a segment or gate descriptor in the LDT; when clear, it indicates that the index refers to a descriptor in the current GDT.

int segment_error_t::segment_selector_index
 

Segment Selector index.

The segment selector index field provides an index into the IDT, GDT, or current LDT to the segment or gate selector being referenced by the error code. In some cases the error code is null (that is, all bits in the lower word are clear). A null error code indicates that the error was not caused by a reference to a specific segment or that a null segment descriptor was referenced in an operation.
